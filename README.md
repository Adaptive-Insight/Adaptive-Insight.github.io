# Adaptive Insight

This repository publishes the canonical Laika product page at
[adaptive-insight.github.io](https://adaptive-insight.github.io/).

The site also publishes the Laika Resources hub and product notes under
[`/resources/`](https://adaptive-insight.github.io/resources/).

Laika installers remain signed GitHub Release assets in
[`Adaptive-Insight/LaikaApp`](https://github.com/Adaptive-Insight/LaikaApp/releases).

## Local Preview

The site is dependency-free and can be served by any static HTTP server:

```bash
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173/`.
